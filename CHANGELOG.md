# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-02-12

### Added
- Initial public release
- Adaptive rolling average algorithm for cycle predictions
- Config flow setup wizard with optional initial cycle data entry
- Binary sensor: Period Active
- Sensors: Current Phase, Cycle Day, Next Period, Period Length, Cycle Length, Fertile Window
- Services: log_period_start, log_period_end, log_symptom
- Real-time entity updates via dispatcher
- Local storage via Home Assistant Store
- HACS support
